# HTML Assignment Webpage

 ## About this assignment:
This is a simple HTML webpage created as part of a college assignment.  
The purpose of this project is to practice basic HTML structure and tags.

 ## The webpage includes sections like:
- About Me  
- Skills  
- Experience  
- Projects  
- Education  
- Contact details  

-----------------------------------------------------------------------------

## Technologies used
- HTML5

-----------------------------------------------------------------------------

## How to run the project:
   No setup is needed.

1. Download or clone this repository
2. Open the project folder
3. Double-click on `index.html`
4. The webpage will open in your browser

-----------------------------------------------------------------------------

 ## How to use
   This is a static webpage.  
   You can scroll through the page to view the different sections.
